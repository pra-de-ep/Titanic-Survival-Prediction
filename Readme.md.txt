# Titanic Survival Prediction 🚢💡

This project uses Machine Learning to predict the survival of passengers on the Titanic using various features like age, gender, class, etc.

## 🔍 Problem Statement

Predict whether a passenger survived or not using historical data from the Titanic disaster.

## 🧠 ML Model Used

- Logistic Regression
- Accuracy Achieved:82%

## 💻 How to Run

1. Clone this repo
2. Install dependencies:  
   `pip install -r requirements.txt`
3. Run the script:  
   `python titanic_model.py`

Displays prediction accuracy, confusion matrix, and survival stats.
